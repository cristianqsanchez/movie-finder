# 📽 React Movie Finder

App to learn React Hooks

## 🧙‍♂️ Instalation


### ⚡ Clone repository
```
git clone https://github.com/cristianqsanchez/movie-finder.git
cd movie-finder
```

### 📦 Install dependencies 
```
pnpm i
```

### 🚀 Run project
```
pnpm run dev
```
